## Objective
The goal of this project is to initialize the Antigravity Engineering workforce. 

This ensures that the CEO properly directs the system architect, the architect plans the stack layout, the engineers can mock up the boilerplate application, and QA can verify the git diff successfully.

### Key Milestones
- **Phase 1 (CEO & Architect):** Agree on the tech stack (e.g. Node + React bundle).
- **Phase 2 (Engineering Leads):** Output a simple "Hello World" or equivalent sandbox implementation.
- **Phase 3 (QA):** Verify the basic code runs without errors.
